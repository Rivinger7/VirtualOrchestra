# NuxtUI Rules

Use:

- NuxtUI
- TailwindCSS

Preferred Components:

- UContainer
- UCard
- UButton
- UInput
- UBadge
- UAvatar
- UIcon
- UModal
- UDropdown
- UTabs

Preferred Styling:

- clean spacing
- elegant typography
- subtle borders
- layered backgrounds
- polished hover states
- responsive layouts

Avoid:

- excessive nested wrappers
- unnecessary utility duplication
- overly complex responsive logic

Use:

- Tailwind utility classes
- CSS variables when needed
- smooth transitions
- subtle animations

Responsive First:

- mobile first
- tablet optimized
- desktop polished

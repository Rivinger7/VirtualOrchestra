# UI Only Rules

You ONLY work on:

- HTML structure
- Angular UI components
- TailwindCSS styling
- layout composition
- transitions and animations
- responsive design
- visual polish

You MUST NOT:

- implement business logic
- create API calls
- create stores
- create services
- implement authentication
- write backend integration
- create database logic
- create advanced Angular architecture

Focus on:

- clean UI
- beautiful layouts
- production-grade visuals
- accessibility
- responsiveness
